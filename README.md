# print_master

## Overview / نظرة عامة

**print_master** is a specialized tool designed to convert UI widgets into images for seamless printing. It allows users to capture widgets, preview them, and export them into various formats including PDF, ZPL, and CPCL for compatibility with different printer types.

**برنت ماستر (print_master)** هو أداة متخصصة مصممة لتحويل عناصر الواجهة (Widgets) إلى صور لطباعتها بسلاسة. يتيح التطبيق للمستخدمين التقاط العناصر، ومعاينتها، وتصديرها إلى تنسيقات مختلفة تشمل PDF و ZPL و CPCL لضمان التوافق مع أنواع الطابعات المختلفة.

---

## Key Features / المميزات الرئيسية

### 1. Widget to Image Conversion / تحويل العناصر إلى صور

- **English:** Convert any UI component or widget into a high-quality image format.
- **عربي:** تحويل أي مكون واجهة أو عنصر (Widget) إلى تنسيق صورة عالي الجودة.

### 2. Multi-Format Export / التصدير بتنسيقات متعددة

- **English:** Generate print-ready files in PDF, ZPL (Zebra Programming Language), and CPCL formats.
- **عربي:** إنشاء ملفات جاهزة للطباعة بتنسيقات PDF و ZPL و CPCL.

### 3. Print Preview / معاينة الطباعة

- **English:** Visualize the converted widget before sending it to the printer to ensure accuracy.
- **عربي:** معاينة العنصر المحول قبل إرساله إلى الطابعة لضمان الدقة.

### 4. Industrial Printer Support / دعم الطابعات الصناعية

- **English:** Native support for label and mobile printers using ZPL and CPCL protocols.
- **عربي:** دعم أصلي لطابعات الملصقات والطابعات المحمولة باستخدام بروتوكولات ZPL و CPCL.

---

## Technical Specifications / المواصفات التقنية

- **Rendering:** High-fidelity widget-to-image processing.
- **Output Formats:** PDF (Documents), ZPL (Label Printers), CPCL (Mobile Printers).
- **Compatibility:** Designed for cross-platform printing workflows.

- **المعالجة:** معالجة عالية الدقة لتحويل العناصر إلى صور.
- **تنسيقات الإخراج:** PDF (للمستندات)، ZPL (لطابعات الملصقات)، CPCL (لطابعات الجوال).
- **التوافق:** مصمم لسير عمل الطباعة عبر الأنظمة المختلفة.
